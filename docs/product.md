# Church Finance AI Manager

## Product Overview
A modern, AI-powered financial management system designed specifically for churches. It combines traditional accounting principles with cutting-edge AI to simplify stewardship, automate data entry, and provide deep financial insights.

## Core Value Proposition
- **Automation**: Reduce manual data entry with AI receipt scanning and automated categorization.
- **Accessibility**: Make financial data accessible to non-accountants via natural language chat.
- **Transparency**: Clear, real-time dashboards and audit rails.

## Technology Stack
- **Frontend**: Next.js 14, React, Tailwind CSS, Shadcn UI
- **Backend/Database**: Supabase (PostgreSQL), Edge Functions
- **AI Engine**: Groq (Llama 3 70B) for reasoning, Google Gemini for Vision & Embeddings
- **Storage**: Supabase Storage for receipts/documents

## Target Audience
- Church Treasurers
- Finance Committees
- Church Administrators
- Pastors (for high-level insights)
